# 牛肉丸粿条汤

本章介绍牛肉丸粿条汤的一般家庭做法。

## 食材

| 食材 | 规格 | 备注 |
| - | - | - |
| 牛肉丸 | 适量[^分割方式] | 可（部分）替换为牛筋丸 |
| 粿条 | 适量 | 可替换为河粉、米粉或面条 |
| 青菜 | 适量 | 推荐枸杞叶、生菜或上海青 |
| [潮汕香芹](../material/celery.md#叶芹) | 适量 | 提供香气，应避免使用西芹等替代 |
| 炸蒜蓉 | 适量 | 可选，提升风味 |

## 步骤

1. 准备所有食材，牛肉丸不需清洗，青菜和芹菜泡淡盐水洗净沥干，芹菜切粒备用，青菜[焯水](../technic/blanching.md)备用
2. 锅中加入适量饮用水，放入牛肉丸，**大火煮沸**后**开盖**，保持**大火**继续煮约 `5` 分钟，后转小火继续煮约 `10` 分钟，直至锅中飘出浓浓的牛肉香气
3. 下入粿条，煮约 `2` 分钟至粿条变软熟透
4. 加入芹菜粒，大火煮约 `1` 分钟后关火，撒上炸蒜蓉，即可出锅享用

[^分割方式]: 在运输和保存过程中难免出现化冻现象，会导致牛肉丸粘连在一起，如果难以分割出需要的数目，可将其平放于厨房案板等硬质平面上，用刀背轻轻敲打牛肉丸间隙，直至分离
